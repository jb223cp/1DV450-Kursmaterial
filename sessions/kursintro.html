Vad är ett webbramverk?

Webbramverk

A web application framework (WAF) is a software framework that is designed to support the development of dynamic websites, web applications, web services and web resources. The framework aims to alleviate the overhead associated with common activities performed in web development. http://en.wikipedia.org/wiki/Web_application_framework

Mjukvaruramverk

Generell och återanvändningsbar kod som stödjer processen att utveckla mjukvara.

  • Kompilatorer för olika programmeringsspråk och plattformar
  • Kod-bibliotek
  • Applikationsramverk
  • Domänramverk
  • Arkitekturramverk (Enterprise architecture framework)

Nyckelbegrepp

  • Inversion of control - Programmets flöde styrs inte direkt av den som anropar koden utan av ramverket
  • Default behavior - Ett ramverk har en "default behaivor" som leder till någon nytta för utvecklingen
  • Extensibility - Ett ramverk kan byggas ut av dess användare antingen genom överskuggning eller egen skriven utökad funktionallitet.
  • Non-modifiable framework code - Ramverkets kod är inte tänkt att bli ändrad i (förutom utökning - extends)

Varför ramverk?

  • Göra repetativa och tråkiga saker mer automatiserat
  • Spara tid?
  • Utprövad "best practice"
  • Implementation av designmönster, filosofi
  • Flera utvecklare, gemensam bakgrund, gemensamma värdegrunder och riktlinjer
  • Fokusera på applikationen === utvecklarglädje!

Foto cc by: http://labautomationrobotics.wordpress.com/

Varför inte ramverk?

  • Behövs det ett ramverk till din applikation?
  • "Det händer massa automagiska saker! Jag vill ha kontroll!"
  • Tappar kontrollen - ramverket står i vägen för mer avancerade lösningar
  • Säkerhet?
  • Prestanda?
  • Versionshelvetet! Beroenden!
  • Tappar förståelse för programmeringslogik?
  • "Det slutar ändå med att man gjort ett eget ramverk"

Hur väljer du ett ramverk?

  • Popularitet
  • Licensformer
  • Inlärningskurva?
  • Support och dokumentation
  • Ramverkets filosofi, designmönster
  • Snabbhet att utveckla
  • Uppdateringsfrekvens, Lever ramverket?
  • Tycke och smak

Ruby gems

  • "Package manager" för Ruby
  • Ett gem är ett paketerat Ruby-bibliotek eller ruby-applikation
  • Rails installeras som ett gem
  • Finns en mängd olika gems som man kan inkludera i sin Ruby-applikation (gem-file) t.ex. kryptering, mailhantering, utökningar av ramverket

  • null is called nil, ett objekt som inte representerar något
