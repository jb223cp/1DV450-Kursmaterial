Vad är ett webbramverk?

Webbramverk

A web application framework (WAF) is a software framework that is designed to support the development of dynamic websites, web applications, web services and web resources. The framework aims to alleviate the overhead associated with common activities performed in web development. http://en.wikipedia.org/wiki/Web_application_framework

Mjukvaruramverk

Generell och återanvändningsbar kod som stödjer processen att utveckla mjukvara.

  • Plattform
  • Kod-bibliotek
  • Applikationsramverk
  • Webbapplikationsramverk
  • Arkitekturramverk (Enterprise architecture framework)

Nyckelbegrepp

  • Default behavior - Ett ramverk har ett "default behaivor" som ska leda till någon nytta för utvecklingen
  • Inversion of control - Programmets flöde styrs inte direkt av applikationsutvecklaren utan av ramverket
  • Non-modifiable framework code - Ramverkets kärnkod (core code) är inte tänkt att bli ändrad i
  • Extensibility - Ett ramverk kan byggas ut antingen genom överskuggning eller egen skriven utökad funktionallitet.

Varför ramverk?

Varför ramverk?

  • Automatisering av återkommande uppgifter
  • Spara tid!?
  • Flera utvecklare, gemensam bakgrund, gemensamma värdegrunder och riktlinjer
  • Utprövad "best practice"
  • Security by default!?
  • Implementation av designmönster, filosofi
  • Fokusera på applikationen === utvecklarglädje!

Varför inte ramverk?

Varför inte ramverk?

  • Behövs det ett ramverk till din applikation?
  • "Det händer massa automagiska saker! Jag vill ha kontroll!"
  • Ramverket står i vägen för mer avancerade lösningar
  • Prestanda?
  • Versionshelvetet! Beroenden!
  • Tappar förståelse för programmeringslogik?
  • "Det slutar ändå med att man gjort ett eget ramverk"

class LinksController < ApplicationController # arv

  def index # frivilligt med paranteser i parameter anrop
      @links = Link.all   # inga semikolon, en sats per rad
  end

  def new
    @link = Link.new # Anrop till kontrukturn via new
  end
  #Inga klammerparanteser, inget krav på tab men standard
  def create
    @link = Link.new(params[:link]) # Hashes och symbols

    if @link.save
      redirect_to links_path # ramverksspecifik metodanrop
    else
      render :action => "new"
    end
  end
end
